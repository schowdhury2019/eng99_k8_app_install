## Basic Commands (windows)

- kubectl version
- kubectl get service
- kubectl get deploy
- kubectl create -f <deployment.yml>
- kubectl create -f <service.yml>
- kubectl delete deploy <deployment>
- kubectl get pods
- kubectl delete pod <pod_name>
